What is it?

Alkitab Bible Study is free desktop based Bible Study tool. It features rich and user friendly Bible study tools. It support parallel views and a powerful search capability. Alkitab is a front end to the "sword project" which is available for Windows, Unix/Linux, etc.

Alkitab requires Java 6 Update 13 or higher (http://www.java.com/getjava).
As a cross platform application, Alkitab runs on any platform that support Java 6 Update 13 or higher
eg: Microsoft Windows, Unix or Linux, etc.

The latest news about Alkitab Bible Study can be found on http://www.kiyut.com/products/alkitab/

System Requirements

  • Java 6 Update 26 or higher (http://www.java.com/getjava)
  • 1024 x 768, 16-bit (thousands of colors) color display or higher
  • Mouse or equivalent pointing device

 

Upgrading to newer version

You need to remove the older version, please use one of the following:

  • Use the "[INSTALL-DIR]\Uninstaller\uninstaller"
  • Or delete the older version folder eg: "C:\Program Files\Kiyut\Alkitab-xxx"

 

Installing Application

To install the product, please do the following:

  • unzip the file
  • on Microsoft Windows double click setup.exe or setup.jar
    on Linux/Unix run setup, if necessary chmod +x setup to make it executable
    on Mac OSX run the Java setup by clicking on the setup.jar
        or you can try the Mac OSX installer (experimental) which is available at http://www.kiyut.com/products/alkitab/download.html
    on other platform type "java -jar setup.jar"
  • follow the on screen instruction

Alternatively, you also could install/run the Alkitab without GUI installer:

  • unzip the file
  • copy the ../packs/Alkitab-xxx into a folder/directory of your choice
  • [Unix/Linux only] run chmod +x Alkitab-xxx/bin/alkitab
  • run or double click Alkitab-xxx/bin/alkitab.exe (Windows) or Alkitab-xxx/bin/alkitab (Unix/Linux)
  • you are done

note: the installer wizard is basically just copying the folder as above into your preferred folder asked by the wizard and create icon shortcut.

 

Launching Application

  • on Microsoft Windows just run Alkitab from Desktop Shortcut or Start Menu - All Programs - Kiyut - Alkitab
    on Mac OSX
    • open the Terminal program (inside Applications folder - Utilities)
    • type "cd [INSTALL-DIR]/bin/" eg: cd /home/user-name/Kiyut/Alkitab/bin/
    • type "./alkitab"
    • or you can just run Alkitab by double click the file alkitab at "[INSTALL-DIR]/bin/alkitab" eg: /Applications/Kiyut/Alkitab/bin/alkitab and choose open with terminal
    on other platform run [INSTALL-DIR]/bin/alkitab, if necessary chmod +x alkitab to make it executable
  • enjoy the product
