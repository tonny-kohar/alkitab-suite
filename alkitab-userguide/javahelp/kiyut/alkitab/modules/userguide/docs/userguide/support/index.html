Support and Feedback

For technical support, bug reports, feature requests, or any feedback, please send email to support@kiyut.com. If possible, please include details of your computer specification* (eg: Operating System, Java VM version, etc), steps to recreate the problem and attach any examples you have where applicable.

note: computer specification information is available from Menu - Help - About.

Your feedback is important to us in order to get an idea of how to make Alkitab better product for you. If you have an idea for a new feature, or a better way of doing something, please drop us a note.
