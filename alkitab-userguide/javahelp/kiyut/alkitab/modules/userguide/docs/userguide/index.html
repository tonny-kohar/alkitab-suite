Overview

Alkitab Bible Study is free desktop based Bible Study tool. It features rich and user friendly Bible study tools. It support parallel view, commentaries, dictionaries, and daily devotions. It also come with powerful search capability.

Alkitab is built using JSword engine and is an alternative front end for Bible Desktop.

Alkitab Bible Study has been placed into the public domain.

Features

  • View Single Book or Bible
  • View Parallel Books or Bibles
  • Cross Reference and Search
  • Commentaries
  • Strong's Concordance and Morphology
  • Lexicons/Dictionaries/Glossaries
  • Daily Devotions
  • Plugins Support
